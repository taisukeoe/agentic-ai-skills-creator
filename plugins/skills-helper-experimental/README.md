# skills-helper-experimental

Experimental skills for AI agent skill development, including testing and iteration workflows.

**Note:** Skills in this plugin are experimental and may change significantly between versions.

## Installation

```bash
/plugin install skills-helper-experimental@agentic-ai-skills-creator
```

## Skills

### running-skills-edd-cycle

Guide evaluation-driven development (EDD) process for agent skills.

**Features:**
- Skill testing workflow setup
- Evaluation scenario creation
- Claude A/B feedback loops for validation
- Development methodology guidance

**Use when:**
- Setting up skill testing workflows
- Creating skill evaluation scenarios
- Establishing feedback loops for skill validation
- Learning EDD methodology for skills
